# Password Strength Checker

## Description
A Python program that evaluates password strength based on:
- Password length
- Uppercase letters
- Digits
- Special characters

## Features
- Weak Password Detection
- Medium Password Detection
- Strong Password Detection

## Technologies Used
- Python

## Example

Input:
```
Karim123!
```

Output:
```
Strong Password
```
